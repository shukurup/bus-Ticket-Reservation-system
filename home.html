A link is the "address" to a document (or a resource) on the web. Hyperlinks, Anchors, and Links In web terms, a hyperlink is a reference (an address) to a resource on the web. Hyperlinks can point to any resource on the web: an HTML page, an image, a sound file, a movie, etc. An anchor is a term used to define a hyperlink destination inside a document. The HTML anchor element , is used to define both hyperlinks and anchors.
